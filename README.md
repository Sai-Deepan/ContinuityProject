# Amaze Services

A modern, student-focused services platform designed to simplify access to essential campus and technical services through a seamless digital experience.

This repository contains the frontend prototype (MVP) of Amaze Services, showcasing the platform's user experience, interface, and service workflow before backend integration.

---

## 🌟 Overview

Amaze Services is built to provide students with quick, reliable, and accessible services through a single platform. Whether it's ordering electronic components, requesting laptop repairs, or seeking technical assistance, the goal is to make everyday student needs easier to manage.

The MVP focuses on delivering a polished and intuitive user experience using mock data while laying the foundation for future backend integration.

## ⚙️ Getting Started

### Clone the repository

```bash
git clone <repository-url>
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

### Preview the production build

```bash
npm run preview
```
